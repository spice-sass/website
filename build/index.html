Installation & Usage

Spice is available for download via npm or bower

npm

npm install spice-sass

Bower

bower install spice-sass

Once installed you can import spice into your main sass file. Note: the import must be relative to your node_modules/bower_components folder dependent on your chosen method of installation. E.g.

@import "../node_modules/spice-sass/src/spice";
